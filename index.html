css chaina la html matra lhekeko

    kei garo chaina hai yo garna follow these things:

    1. paila create euta normal repository
    2. upload timro website ko file haru (html, css, javascript) wala file
    3. ani go tyo repository ko settings ma
    4. ani there are options haru left ma go to PAGES wala option
    5. you will see branch bhanne option bold ma lhekeko tesko muni default ma (none) selected huncha
    6. change that to (main) dropdown menu bata select garera
    7. ani (main) banayeshi aarko dropdown menu ma pailai dekhi (root) cha bhani okii natra tyo option ma select root
    8. ani save ko option cha bheni SAVE ni feri
    9. reload tyo page
    10. you will see website ko link ra website ezz
    try la

    p.s. website ma dekhauna halka time lhagcha la don't panic
    you might think aaba phishing page yesmai banauchu try it mildaina
